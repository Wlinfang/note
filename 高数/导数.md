* R-实数的证明

```
1、Dedekind + Cantor 证明

2、分割概念

   一个集合，分为 两个非空集合 A,B； 假设 对任一 a ∈ A，b ∈ B，有 a < b 

   A 叫做 分割的下类   B 叫做 分割的上类   A|B 表示 


   情况1  如果 A 为 a < 1 的 所有数的集合； B 为 b >= 1 的所有数的集合；A中无最大值，B中有最小值,有理数分割

   情况2  如果 A 为 a<= 1 的 所有数的集合；B 为 b > 1 的所有数的集合；A中有最大值，B中无最小值，有理数分割

   情况3  如果 A 为 c^2 < 2 的 所有数的集合； B 为 c^2 > 2 的 所有数的集合; A中无最大值, B中无最小值，无理数分割

         这种情况引入了无理数

   情况4  不可能A 中 有最大值，B中有最小值同时存在； 原因是 必然存在 c 使  c > A; c < B
```

* 没有一个其平方能=2 的有理分数p/q 存在

```
1、p,q 为 自然数

2、反正法

3、如果存在，p,q ,且假设 p,q 无公因子， (p/q)^2 = 2 

   则  p^2=2q^2 , 则 p为偶数，q为奇数，因为无公因子

   设 p=2r, 则 q^2 = 2r^2 则 q 为偶数，同之前 q 为奇数 矛盾
```

* 确界 

```
如果集合 A 中 有上界，则 必然 存在 上确界

如果集合 A 中 有下界，则 必然 存在 下确界


证明：

情况1  假设 存在 A 中 最大值 a,且 a ∈ A ,则 对于集合A 中任一 均 <= a, 对任一上界 M ，有 a <= M 

情况2  假设 A 不存在最大值 a ，则 根据 实数分割，可以分为  A 集合 和 A 的上界集合 M ，
      
      其中 A 中的任一 均 < M 中的任一数； 根据实数分割原理，一定有分割的界数 β 存在
      
      则 β 为 上界M 中的 最小值，即为 上确界
```
