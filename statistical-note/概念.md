* 监督学习

```
监督学习模型分为 概率型、非概率型

概率型 使用条件概率  P(Y|X) 表示

非概率型 使用决策函数 Y=f(X) 表示
```

* 损失函数

```
L  预测模型一次的好坏

分为以下几种

1、0-1 损失函数

  L =  1， Y=f(X) ， L=0 ，Y ≠ f(X)

2、平方损失函数

   L =  (Y- f(X))^2 

3、绝对值损失函数

    L = | Y -  f(X) |

4、对数损失函数

    L = -logP(Y|X)
```

* 风险函数

```
预测模型平均意义上的好坏

即 损失函数的期望

∫L(Y,f(X))P(x,y)dxdy  区间为 X✘Y
```
